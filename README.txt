These instructions assume you use VSCode.  
1. If you dont already have it installed - install VSCode
2. Navigate to: https://code.visualstudio.com/docs/languages/java
   a. Click the "Install the Coding Pack for Java"  to download.
   b. Follow the prompts to install the Coding Pack
   c. On the same page, scroll down and click "Install the Extension Pack for Java"
   d. This will open VSCode - Follow the prompts to install (This may already be installed)
3. Press F5 to run the application
4. In your browser, navigate to http://localhost:8080/gametype
   a. the json data for the classic game type should be displayed.
